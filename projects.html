Daily Joy

GitHub

A SwiftUI iOS app that helps users build a gratitude habit by saving meaningful moments, notes, photos, streaks, and memories.

Tech Stack

SwiftUI, SwiftData, iOS, accessibility APIs, local animations.

Features

Moment collection, searchable history, streak calendar, achievements, privacy locking, onboarding, and celebration effects.

Challenge

Balancing a rich emotional experience with organized app state, persistence, privacy UI, and reusable SwiftUI components.

Lesson Learned

Small details like motion, onboarding, and accessibility labels can make a simple app feel much more complete.

Complete / MECO Robotics

Pit Scheduler

GitHub

A Google Apps Script and Google Sheets scheduler for robotics teams that posts shift schedules to Slack and sends individual reminders.

Tech Stack

Google Apps Script, Google Sheets, HTML, CSS, JavaScript, Slack API.

Features

Shift CRUD, CSV bulk import, Slack channel posts, DM reminders, printable schedules, and live on-duty panels.

Challenge

Coordinating spreadsheet data, time-based triggers, Slack user mapping, and a simple web UI in one Apps Script project.

Lesson Learned

Operational tools need clear data validation, dependable notifications, and workflows that are quick during busy events.

MECO Mission Control Mobile

GitHub

An Expo and React Native mobile client for robotics manufacturing, planning, meetings, task visibility, and operations workflows.

Tech Stack

Expo, React Native, React, TypeScript, ESLint, EAS release automation.

Features

Subsystem dashboard, task workflow states, meeting RSVP and attendance, work-log visibility, manufacturing queues, and QA outcomes.

Challenge

Designing a mobile interface that can represent complex team operations without overwhelming students, mentors, or admins.

Lesson Learned

Large tools work best when data, roles, and UI states are planned early instead of added after screens are built.
